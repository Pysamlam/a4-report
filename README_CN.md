## 这是什么

**A4 Report** 是一个 Claude Code 技能（Skill），能将你的主题或数据转化为一份精致的、可打印的咨询报告。输出结果是**单个独立 HTML 文件**——无需 Node.js、无需 Python、无需构建工具。浏览器打开 → 打印 → 另存为 PDF，三步完成。

**典型使用场景：**
- 市场分析与竞品研究
- 战略规划与可行性研究
- 业务复盘与董事会汇报
- 客户提案与尽职调查摘要

## 快速开始

### 第一步 — 安装

**方式 A：插件市场**
```bash
/plugin marketplace add samgor/a4-report
/plugin install a4-report@a4-report
```

**方式 B：手动安装**
```bash
mkdir -p ~/.claude/skills/a4-report
cp SKILL.md STYLE_PRESETS.md a4-report-template.md ~/.claude/skills/a4-report/
```

### 第二步 — 调用

在 Claude Code 中输入 `/a4-report`，然后描述你的需求：

```
/a4-report

帮我写一份中国新能源汽车市场分析报告，面向董事会，标准版。
```

```
/a4-report

Write a competitive intelligence report on the top 3 cloud providers,
McKinsey style, 10-15 pages, for internal strategy review.
```

### 第三步 — 获取报告

Claude 会引导你完成简短的确认问答，生成 HTML 文件并自动打开。在浏览器中打印导出 PDF 即可。

## 工作原理 — 七阶段流程

```
阶段 1：内容确认    → 汇报对象、报告类型、目的、篇幅
阶段 2：风格选择    → 预览三种风格或直接选择
阶段 3：报告生成    → 完整 HTML，内联 CSS，零依赖
阶段 4：图表增强    → 每张图表自动添加读图指引 + 关键洞察
阶段 5：交付        → 浏览器打开，随时打印
阶段 6：PDF 导出    → 分步骤浏览器打印指引（可选）
阶段 7：数据审计    → 自动校验报告中每一个数字（有源数据时触发）
```

技能只问必要的问题，然后不间断地生成完整报告。

## 使用指南

### 基础用法 — 纯主题

最简单的方式：给出主题，选择风格，获得报告。

```
/a4-report

帮我写一份东南亚即时零售市场进入报告。
```

Claude 会依次确认：
1. **汇报对象** — 谁来读这份报告？（管理层 / 董事会 / 业务负责人 / 咨询顾问）
2. **报告类型** — 市场分析 / 竞品研究 / 战略规划 / 业务复盘等
3. **报告目的** — 投资决策 / 董事会汇报 / 客户提案等
4. **篇幅** — 简洁版（5-8 页）/ 标准版（10-15 页）/ 详细版（20 页以上）
5. **风格** — 麦肯锡蓝 / BCG 银灰 / 贝恩红（可预览后选择）

### 进阶用法 — 携带源数据

上传或引用数据文件，技能将自动激活**数据审计**（阶段 7）：

```
/a4-report

基于附件的 Excel 数据，帮我写一份美团 2025 年财务复盘报告，
面向管理层，麦肯锡风格，标准版。
```

报告生成后，阶段 7 自动执行：
- 提取报告中的每一个数字
- 与源数据文件逐一核对
- 标记疑似幻觉 ⚠️ 和确认错误 ❌
- 显示准确率，并提供一键自动修复

**审计输出示例：**
```
╔══════════════════════════════════════════════════╗
║              🔍  数据审计开始                     ║
╠══════════════════════════════════════════════════╣
║  报告文件: meituan-review-2025.html              ║
║  源数据文件: data.xlsx                           ║
╚══════════════════════════════════════════════════╝

✅ 已验证:    12 项
⚠️ 疑似幻觉:   2 项  （需要人工确认）
❌ 数据错误:   1 项  （需立即修正）

📊 准确率: 80%
```

### 风格选择

三种预设风格，每种对应不同的顶级咨询公司美学：

| 风格 | 视觉特点 | 适用场景 |
|------|---------|---------|
| **麦肯锡蓝** | 深海军蓝 + 白色，衬线标题（Playfair Display） | 高管汇报、正式报告 |
| **BCG 银灰** | 深灰 + 几何布局，简洁无衬线（Inter） | 结构化分析、战略材料 |
| **贝恩红** | 大红强调色，冲击力数字（DM Serif Display） | 数据密集型报告、业绩复盘 |

选择"预览风格"时，Claude 会生成三份单页 HTML 预览，对比后再做决定。

## 核心特性

### 零依赖输出
每份报告都是**单个 HTML 文件**，所有 CSS 内联。无需 npm、无需 Python、无需构建步骤。有浏览器的设备都能使用。

### 图表注解系统
每张图表自动附带：
- **📊 读图指引** — 解释坐标轴、颜色含义和关注重点
- **💡 关键洞察** — 趋势判断、对比发现、行动启示

告别"看不懂图表"的困境。

### 打印优化排版
- 精确 A4 格式：210mm × 297mm
- `@page { size: A4; }` 媒体查询，确保 PDF 渲染完美
- 智能分页——标题不会孤悬于页面底部
- 打印模式保留背景图形

### 受众自适应内容
提前选定汇报对象，报告内容自动调整：
- **管理层 / C-Suite** → 结论先行，数据驱动，聚焦决策
- **董事会 / 投资者** → 强调 ROE、现金流、战略价值
- **业务负责人** → 操作层面分析，可执行建议
- **外部客户** → 价值主张，合作亮点

### 咨询级报告结构
每份报告遵循标准咨询报告层级：
1. 封面页
2. 执行摘要（3-5 条核心结论）
3. 目录
4. 关键发现（标题 + 证据 + 启示）
5. 详细分析（图表、框架、数据）
6. 战略建议（优先级排序，可执行）
7. 附录（数据表格、方法论说明）

## 导出为 PDF

1. 用 Chrome 或 Safari 打开生成的 HTML 文件
2. 按 `Cmd+P`（Mac）或 `Ctrl+P`（Windows）
3. 目标打印机 → 选择 **另存为 PDF**
4. 纸张大小 → **A4**
5. 勾选 **背景图形**
6. 保存

报告内置打印按钮（打印 / Save as PDF），点击直达打印对话框。

## 文件结构

```
a4-report-main/
├── SKILL.md                  # 核心工作流 — 始终加载
├── STYLE_PRESETS.md          # 三种咨询风格 — 阶段 2 加载
├── a4-report-template.md     # A4 HTML 结构 — 阶段 3 加载
├── scripts/
│   ├── audit-checklist.md    # 数据审计指南 — 阶段 7 加载
│   └── export-pdf.sh         # PDF 导出参考
└── .claude-plugin/
    ├── marketplace.json
    └── plugins/a4-report/
```

技能采用**渐进式加载**设计——每个阶段只加载当前所需文件，保持上下文精简，生成速度快。
